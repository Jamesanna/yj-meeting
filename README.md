# 羿鈞科技會議室預約系統 (Meeting Hub)

![Version](https://img.shields.io/badge/Version-2.200-orange)
![Security](https://img.shields.io/badge/Auth-Google_Workspace-blue)
![Database](https://img.shields.io/badge/DB-Firebase_Firestore-yellow)

這是一款專為羿鈞科技設計的企業級會議室預約管理系統。系統核心目標在於解決會議室搶用問題，並提供安全、高效且支援行動端的預約體驗。

---

## 🌟 系統亮點 (V2.200)

- **月度預約總覽**: 前台與後台皆具備「當月預約清單」，行程資訊一目了然，不再侷限於單日查看。
- **全平台響應**: 針對手機端深度優化，後台管理介面視圖流暢，支援卡片化與表格切換。
- **管理員直覺維護**: 後台行程清單支援點選直接「修改」或「刪除」，大幅提升管理效率。
- **身分驗證 2.0**: 支援 Google Workspace 實名登入與 RBAC 權限控管（一鍵直達自動登入）。
- **雲端同步**: 全面對接 Firebase Firestore，實現跨裝置即時資料傳輸。

## 📅 完整版本更新紀錄 (Changelog)

### v2.200 - 月度預約總覽清單上線 (2026-02-09)
- 前台與後台月曆下方新增「本月預約總覽」清單，改採跨日排序顯示，方便查看整月行程。
- 後台月度清單支援管理員直接「點選修改」與「一鍵刪除」功能。
- 優化清單日期標示，加入星期與月份縮寫，提升行動端閱讀體驗。

### v2.199 - 管理員穿透式登入優化 (2026-02-09)
- 實現「一鍵直達」：若已透過 Google 登入且具備管理權限，點擊齒輪將直接進入後台。
- 刷新網頁後，系統會自動恢復 Google Admin 帳號的管理員權限。

### v2.198 - 登入驗證邏輯修復 (2026-02-09)
- 修復「管理權限帳號」無法通過前台 Google 登入白名單驗證之異常。
