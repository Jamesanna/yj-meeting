# 羿鈞科技會議室預約系統 (Meeting Hub)

![Version](https://img.shields.io/badge/Version-2.199-orange)
![Security](https://img.shields.io/badge/Auth-Google_Workspace-blue)
![Database](https://img.shields.io/badge/DB-Firebase_Firestore-yellow)

這是一款專為羿鈞科技設計的企業級會議室預約管理系統。主要目標在於解決會議室搶用問題，並提供安全、高效且支援行動端的體驗。

---

## 🌟 系統亮點 (V2.199)

- **全平台響應**: 針對手機端深度優化，後台管理介面視圖流暢。
- **身分驗證 2.0**: 採用 Google Workspace 帳號比對機制，並支援 RBAC 權限控管系統。
- **內網辨識 (Express Mode)**: 自動偵測辦公室網路環境，縮短預約路徑。
- **雲端同步**: 全面對接 Firebase Firestore，實現即時資料同步。
- **一鍵直達後台**: 具備管理權限的人員，登入 Google 後可免密碼直達管理介面。

---

## 📖 操作指南摘要

### 1. 一般員工
- **身分登入**: 點擊右上方「登入」完成 Google 驗證。
- **快速預約**: 點擊月曆日期或掃描現場 QR Code 即可進行預約。

### 2. 管理端維護
- **權限進入**: 具管理員角色的帳號，登入 Google 後點擊「齒輪」即可進入，不需額外帳密。
- **人員管理**: 必須在此預錄員工 Email，並可在此指派帳號角色。
