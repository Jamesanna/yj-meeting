# 羿鈞科技會議室預約系統 (Meeting Hub)

![Version](https://img.shields.io/badge/Version-2.201-orange)
![Security](https://img.shields.io/badge/Auth-Google_Workspace-blue)
![Database](https://img.shields.io/badge/DB-Firebase_Firestore-yellow)

這是一款專為羿鈞科技設計的企業級會議室預約管理系統。致力於提供直覺、安全且符合台灣使用習慣的辦公空間管理體驗。

---

## 🌟 系統亮點 (V2.201)

- **語系在地化**: 全系統週次顯示優化為中文（如：週一、週二），更符合直覺。
- **管理權限分級**: 劃分「最高管理員 (L1)」與「系統管理員 (L2)」，強化核心帳號安全性。
- **權限穿透限制**: 次管理員無法編輯或刪除最高管理員帳號，確保系統控制權不偏移。
- **月度預約總覽**: 前台與後台皆具備「當月預約清單」，行程資訊一目了然。
- **全平台響應**: 手機端深度優化，支援卡片化管理與一鍵直達自動登入。

## 📅 版本更新紀錄 (Changelog)

### v2.201 - 語系在地化與權限位階強化 (2026-02-09)
- 週次顯示在地化：前台與後台的行程清單，週次由英文縮寫改為中文。
- 管理權限分級：區分「最高管理員 (Level 1)」與「次管理員 (Level 2)」。
- 安全權限限制：位階二帳號 (L2) 無法編輯、修改或刪除位階一 (sysop) 帳號。

### v2.200 - 月度預約總覽清單上線 (2026-02-09)
- 前台與後台月曆下方新增「本月預約總覽」清單。
- 後台月度清單支援管理員直接「點選修改」與「一鍵刪除」功能。

### v2.199 - 管理員穿透式登入優化 (2026-02-09)
- 實現「一鍵直達」：具備管理權限的人員，登入 Google 後可免密碼直達管理介面。
