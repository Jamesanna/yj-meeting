# 羿鈞科技會議室預約系統 (Meeting Hub)

![Version](https://img.shields.io/badge/Version-2.208-orange)
![Security](https://img.shields.io/badge/Auth-Google_Workspace-blue)
![Database](https://img.shields.io/badge/DB-Firebase_Firestore-yellow)

這是一款專為羿鈞科技設計的企業級會議室預約管理系統。致力於提供直覺、安全且符合台灣使用習慣的辦公空間管理體驗。

---

## 🌟 系統亮點 (V2.208)

- **管理員密碼維護**: 修復介面整合後的欄位顯示邏輯，支援 L1/L2 管理員同步重設登入密碼。
- **介面架構整合**: 將「系統帳號」功能併入「員工管理」模組，作為獨立子頁籤。
- **雙軌權限位階管理**: 提供「最高管理員 (L1)」與「系統管理員 (L2)」選項。

---

## 📅 版本更新紀錄 (Changelog)

### v2.208 - 管理員密碼維護功能修正 (2026-02-09)
- 修復密碼欄位顯示：解決介面整合後，因標籤識別代碼變動導致管理員密碼欄位消失的問題。
- 全面支援 L1/L2 密碼重設：無論是 L1 或 L2，只要被指派為管理職位即可設定登入密碼。

### v2.207 - 模組化開發：介面架構整合 (2026-02-09)
- 整合權限管理入口：將原先獨立的「系統帳號」標籤併入「員工管理」模組。
