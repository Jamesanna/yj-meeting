# 羿鈞科技會議室預約系統 (Meeting Hub)

![Version](https://img.shields.io/badge/Version-2.172-orange)
![Security](https://img.shields.io/badge/Auth-Google_Workspace-blue)
![Database](https://img.shields.io/badge/DB-Firebase_Ready-yellow)

這是一款專為羿鈞科技設計的企業級會議室預約管理系統。系統核心目標在於解決會議室搶用問題，並提供安全、高效且支援行動端的預約體驗。

---

## 🌟 系統亮點 (V2.172)

- **全平台響應**: 針對手機端進行深度優化，後台管理介面全面卡片化，操作流暢不卡頓。
- **身分驗證 2.0**: 採用 Google Workspace 帳號比對機制，僅限白名單內的員工可進行預約。
- **內網辨識 (Express Mode)**: 自動偵測辦公室網路環境，縮短預約路徑。
- **假日自動鎖定**: 內建 2026 年台灣行政院辦公日曆，自動跳過國定假日與週末。
- **環境變數架構**: 關鍵參數（SSID、密碼、網域）全面採用 `process.env` 管理，提升安全性。

## 📖 一般員工操作指南

### 1. 會議室預約流程
1. **身分登入**: 點擊右上方「登入」按鈕，完成 Google 帳號驗證。
2. **日期選取**: 在月曆中點擊欲預約的工作日（週一至週五）。
3. **確認時段**: 
   - 系統會彈出預約視窗，預設時段為 30 分鐘一組。
   - 可點擊「快速設定」選取：全日、上午或下午。
4. **填寫用途**: 輸入會議主題或目的。
5. **儲存確認**: 提交後系統會即時更新月曆上的標記，並同步顯示於下方的「預約詳情」。

### 2. 快速預約模式 (Express Mode)
- **掃碼預約**: 掃描會議室門口的 QR Code 進入系統。
- **自動偵測**: 若身處辦公室內網，系統將自動跳過日期選取步驟，直接開啟當日預約視窗。

### 3. 來賓 WiFi 連線
- 點擊標題列的 WiFi 圖示，可查看目前對外開放的 **Guest SSID** 與 **密碼**。
- 提供連線專用 QR Code，手機掃描後可自動連接。

## 🛡️ 管理端維護指南 (Sysop Only)

### 1. 進入管理後台
- 點擊標題列最右方的齒輪圖示（Settings）。
- 輸入最高權限帳號：`sysop` / 密碼：`Aa123456`。
- **建議**: 登入後可勾選「記住此帳號」以便後續快速維護。

### 2. 人員與白名單管理
- **員工管理**: 必須在此預錄員工的 Google Email，同仁方可進行前台預約。
- **部門設定**: 定義公司組織架構，方便在名單中快速篩選人員。

### 3. 公告與系統維護
- **公告管理**: 發佈的訊息會即時顯示在前台頂部的跑馬燈。
- **系統資訊**: 在此檢視改版紀錄，並可動態調整「允許網域」、「WiFi SSID」等核心參數。

## ⚙️ 技術配置與部署

### 1. 環境變數說明 (Environment Variables)
部署時請務必設定以下變數，系統將自動映射至對應功能：

| 變數名稱 | 說明 | 範例值 |
| :--- | :--- | :--- |
| `ALLOWED_DOMAIN` | 限制預約者的電子郵件後綴網域 | `yi-jun.com` |
| `CORP_SSID` | 辦公室內部網路名稱 (SSID) | `.YIJUN` |
| `GUEST_SSID` | 供訪客使用的無線網路名稱 | `.YJ_VIP` |
| `GUEST_PWD` | 訪客 WiFi 密碼 | `@1681688` |
| `API_KEY` | 預留給 Firebase 資料庫的連線金鑰 | `AIzaSy...` |

### 2. Firebase 資料庫對接
本系統目前為 **V2.172 (Firebase Ready)**。
- 服務層已整合至 `services/mockDb.ts`。
- 切換雲端資料庫時，只需將 `localStorage` 調用更換為 `Firestore API` 即可完成無痛遷移。
